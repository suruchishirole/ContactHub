# ContactHub
<i>Smart Contact Management System</i>

<br>
ContactHub is a modern Java desktop application designed to simplify contact management through an intuitive and visually appealing interface.<br>
Built using Java Swing for the frontend and MySQL for data storage, it enables users to efficiently organize, search, and manage personal and professional contacts. It combines a clean, responsive interface with robust database operations, allowing users to manage contact information efficiently.<br>
This project demonstrates practical implementation of Java GUI development, OOP, JDBC connectivity, SQL database management, & user input validation in a real-world desktop application.<br><br>

<b>Features</b><br>
➕ Add, edit, and delete contacts <br>
🔍 Real-time search by name, phone number, or email <br>
🏷️ Category-based organization (General, Family, Friends, Work, VIP) <br>
📊 Dashboard displaying contact statistics <br>
✅ Input validation for phone numbers and email addresses <br>
🕒 Automatic creation and update timestamps <br>
🎨 Modern dark-themed user interface with custom-styled components <br>
💾 Reliable MySQL database integration for persistent storage <br><br>

<b>Technologies Used</b><br>
- Java <br>
- Java Swing <br>
- MySQL <br>
- JDBC <br>
- NetBeans IDE <br>
