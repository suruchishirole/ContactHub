# ContactHub
Smart Contact Management System

ContactHub is a modern Java desktop application designed to simplify contact management through an intuitive and visually appealing interface. 
Built using Java Swing for the frontend and MySQL for data storage, it enables users to efficiently organize, search, and manage personal and professional contacts. It combines a clean, responsive interface with robust database operations, allowing users to manage contact information efficiently.
This project demonstrates practical implementation of Java GUI development, OOP, JDBC connectivity, SQL database management, & user input validation in a real-world desktop application.

<b>Features</b>
➕ Add, edit, and delete contacts
🔍 Real-time search by name, phone number, or email
🏷️ Category-based organization (General, Family, Friends, Work, VIP)
📊 Dashboard displaying contact statistics
✅ Input validation for phone numbers and email addresses
🕒 Automatic creation and update timestamps
🎨 Modern dark-themed user interface with custom-styled components
💾 Reliable MySQL database integration for persistent storage

<b>Technologies Used</b>
Java
Java Swing
MySQL
JDBC
NetBeans IDE
